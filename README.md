# Website CEE

Website da CEE baseado em Django e React. O projeto mira a criação de um site com base de dados interna acessível a alunos do Instituto Tecnológico de Aeronáutica, e a automatização de uma série de processos, como, por exemplo, a publicação de uma vaga.
